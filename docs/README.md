# Dokumentation

Denna mapp ar avsedd for overgripande dokumentation som inte hor till ett enskilt paket.

## Exempel pa innehall

- Arkitektur- och designbeslut
- Contributor-guide och processer
- Release-notes och versionsstrategi
- Integrationsguider for konsumentappar

## Riktlinjer

- Paket-specifik dokumentation ligger kvar i respektive packages/<namn>/README.md
- Storybook-relaterad dokumentation ligger i apps/storybook/README.md
- Hall dokumentationen kort, konkret och uppdaterad nar beteenden eller scripts andras
